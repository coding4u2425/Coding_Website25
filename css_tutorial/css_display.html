CSS Display

The CSS display property is used to control the layout of an element. It determines how an HTML element should be displayed on the webpage.

Here are some common values for the display property:

Display Inline

The inline value allows the element to act like an inline element. It only takes up as much space as necessary.

Example:

HTML

<!DOCTYPE html>
<html>

<head>
    <title> CSS Display </title>
</head>

<body>
   <div class="element1"> Element1 </div>
    <div class="element2"> Element2 </div>
    <div class="element3"> Element3 </div>
 </body>
</html>

CSS

.element1 {
    display: inline;
    background-color: aqua;
}

.element2 {
    display: inline;
    background-color: yellow;
}

.element3 {
    display: inline;
    background-color: green;
}

Output

Display Block

The block value allows the element to behave like a block-level element. It takes the full width available in the webpage.

Example:

HTML

<!DOCTYPE html>
<html>

<head>
    <title> CSS Display </title>
</head>

<body>
   <div class="element1"> Element1 </div>
    <div class="element2"> Element2 </div>
    <div class="element3"> Element3 </div>
 </body>
</html>

CSS

.element1 {
    display: block;
    background-color: aqua;
    height: 100px;
    width: 300px;
}

.element2 {
    display: block;
    background-color: yellow;
    height: 100px;
    width: 300px;
}

.element3 {
    display: block;
    background-color: green;
    height: 100px;
    width: 300px;
}

Output

Display Inline Block

The inline-block value combines characteristics of both block and inline elements. It supports width and height like a block-level element but flows like an inline-level element.

Example:

HTML

<!DOCTYPE html>
<html>

<head>
    <title> CSS Display </title>
</head>

<body>
   <div class="element1"> Element1 </div>
    <div class="element2"> Element2 </div>
    <div class="element3"> Element3 </div>
 </body>
</html>

CSS

.element1 {
    display: inline-block;
    background-color: aqua;
    height: 100px;
    width: 100px;
}

.element2 {
    display: inline-block;
    background-color: yellow;
    height: 100px;
    width: 100px;
}

.element3 {
    display: inline-block;
    background-color: green;
    height: 100px;
    width: 100px;
}

Output

Display None

The none value hides the element from the page. The element is not displayed and does not occupy any space.

Example:

HTML

<!DOCTYPE html>
<html>

<head>
    <title> CSS Display </title>
</head>

<body>
   <div class="element1"> Element1 </div>
    <div class="element2"> Element2 </div>
    <div class="element3"> Element3 </div>
 </body>
</html>

CSS

.element1 {
    display: block;
    background-color: aqua;
    height: 100px;
    width: 200px;
}

.element2 {
    display: none;
    background-color: yellow;
}

.element3 {
    display: block;
    background-color: green;
    height: 100px;
    width: 200px;
}

Output
